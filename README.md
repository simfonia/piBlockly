# πBlockly

This is a VS Code extension for visual programming with Blockly, specifically designed for Arduino and the PiCar.

## Features

*   Visual programming using Blockly.
*   Generate Arduino C++ code.
*   Integrates with the Arduino IDE 2.

## Quick Start

1.  Open an Arduino file (`.ino`).
2.  Click the πBlockly icon in the editor's title bar to open the Blockly editor.
3.  Drag and drop blocks to create your program.
4.  The Arduino code will be automatically updated in the editor.
